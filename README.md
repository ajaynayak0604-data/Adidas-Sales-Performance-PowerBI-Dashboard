# Adidas Sales Performance Analysis – Power BI Dashboard
## Project Overview
This project presents an interactive Power BI dashboard designed to analyze Adidas sales performance across different regions, product categories, and time periods. 

The dashboard helps identify key trends, top-performing products, and regional sales patterns. It transforms raw sales data into meaningful insights that support data-driven business decisions.
## Tools & Technologies
- Power BI
- Data Visualization
- Data Analysis
- Dashboard Design
## Key Insights
- The West region generates the highest overall revenue.
- Men's Street Footwear is the top-performing product category.
- Sales peak during certain months, showing seasonal demand patterns.
- Some regions contribute significantly less revenue, highlighting potential growth opportunities.
## Dashboard Features
- Interactive filters for region, product category, and time
- KPI cards displaying total sales, profit, and units sold
- Sales trend analysis over time
- Regional and product-level performance comparison
